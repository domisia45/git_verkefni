Git spurningar og svör
1. Hvað gera eftirfarandi Linux skipanir?
	cd > Vísar í möppuna
	cd ~ > Vísar ínn í drifið þitt
	ls > sýnir skránna
	pwd > prentar fullan aðgang í hverju ertu núna
	mkdir > býr til skrá

	2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone > Ef þú vilt fá afrit af núverandi Git geymslu
	git log > sýnir upplýsinga um fyrri skuldbindingar
	git status > sýnir stöðu vinnustaðsins
	git diff > sýnir stöðu á milli vinnustaðsins og skuldbindingum
	git checkout > skipta yfir útibú eða endurheimtir skrár

	3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff > sýnir breytingar á skuldbindingum og vinnusvæði
	b) git diff  --staged > sýnir breytinga á skyndiminni/indexinu
	c) git diff commit1 commit2 > sýnir muninn á þeim skrá

	4. 	Hvað er útgáfustýring (e.version control)? >  geymir allar útgáfur af verkefni og leyfir manni að fara á milli þeirra

	5.	Hverjir eru helstu kostir við að nota GIT? > Að vinna með fólki á 2 tölvum

	6.	Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu? > Eins oft og það getur og það  hægir ekki á verkferli manns

	7.	Hvað er "Working directory"?" "Staging area"?" og "Repository" í GIT? > Working directory er núverandi vinnusvæði,Staging area er skrá sem geymir upplýsingar um verkefnið og Repository er verkefna mappan

	8.	Hvenær er sniðugt að nota greinar (branches)? > Þegar þú ert ekki viss um breytingarnar sem þú ætlar að breyta

	9.	Hver fann upp GIT? > Linus Benedict Torvalds fann up
